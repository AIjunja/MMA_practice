# 병역명문가 혜택찾기 디자인 명세

## 제품 목적

병역명문가와 가족이 경기도 지역과 업종을 선택해 예우시설의 우대 대상, 감면 내용, 비고와 연결 가능한 연락 정보를 확인하는 모바일 우선 웹앱이다.

## 핵심 흐름

1. 지도 또는 지역 이름으로 시군 선택
2. 교육, 문화, 숙박&관광, 의료, 주차, 체육, 기타 중 하나 선택
3. 시설명 또는 감면 내용 검색
4. 상세 화면에서 우대 대상, 감면 내용, 비고 확인
5. 원본에 있는 경우에만 전화, 홈페이지, 네이버 지도 연결

## 시각 방향

- 차분한 공공 안내서와 종이 지도의 중간 인상
- Gmarket Sans를 제목과 본문에 사용
- 짙은 남색 `#123665`를 신뢰 색상으로 사용
- 금색 `#C89228`은 진행 상태와 강조에만 제한적으로 사용
- 둥근 카드를 반복하지 않고 구분선과 정보 위계로 목록을 구성
- 장식용 그라데이션, 글래스모피즘, 과도한 모션 사용 금지

## 데이터 표시 원칙

- 시설명, 우대 대상, 감면 내용과 비고는 의미를 바꾸지 않는다.
- 비고의 빈칸과 대시는 표시하지 않는다.
- 주소, 전화번호, 홈페이지가 있을 때만 해당 행동 버튼을 표시한다.
- 주소가 없는 시설을 임의의 지역 중심점에 표시하지 않는다.
- `경기도` 관리 시설은 주소에서 실제 위치 시군을 분리해 지역 검색에 연결한다.
- 실제 단일 장소가 아닌 시설군과 지역 공통 혜택은 안내 문구를 함께 보여준다.
- 지도는 통계청 SGIS 기반의 실제 시군 경계를 사용하고, 일반구가 있는 도시는 하나의 선택 영역으로 동작한다.
- 사진은 시설명이 정확히 일치하고 재사용 라이선스와 저작자를 확인할 수 있을 때만 연결한다.
- 사진 원본과 라이선스 링크는 상세 화면에서 항상 함께 제공한다.
- 사진이 없는 시설은 업종 아이콘을 사용하며 임의의 유사 장소 사진으로 대체하지 않는다.

## 반응형 기준

- 320px 이상 모바일 지원
- 700px 이하에서는 지역 지도와 지역명 목록을 토글로 전환
- 940px 이하에서는 카테고리 카드를 2열 이하로 축소
- 상세 화면은 모바일에서 이미지 영역과 정보를 세로로 배치
- 모든 버튼은 최소 42px 이상의 터치 영역을 확보

## 접근성

- 본문 바로가기 제공
- 화면 전환 후 새 화면 제목에 포커스 이동
- 키보드 포커스에 3px 금색 외곽선 표시
- 색상만으로 면제와 할인 상태를 구분하지 않음
- 지도 사용이 어려운 사용자를 위한 전체 지역명 목록 제공
- `prefers-reduced-motion`에서 애니메이션 제거

## 모션

- 화면 전환: 440ms 이내의 투명도와 수직 이동
- 업종 카드: 카드 사이 35ms 간격의 한 번만 실행되는 등장 모션
- hover: 위치 이동 4px 이하
- 자동 재생, 패럴랙스, 반복 애니메이션 금지
