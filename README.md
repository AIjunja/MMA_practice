# 병역명문가 혜택찾기

경기도 병역명문가 예우시설을 지역과 업종별로 찾는 모바일 우선 웹앱입니다.

## 주요 기능

- 통계청 SGIS 행정경계를 바탕으로 한 경기도 31개 시군 SVG 지도
- 혜택 자료가 있는 30개 시군 선택과 자료가 없는 지역 구분
- 7개 업종 필터
- 시설명·감면 내용 검색
- 우대 대상, 감면 내용, 비고 상세 조회
- 원본 데이터가 있는 경우 전화·홈페이지·네이버 지도 연결
- 주소가 없는 시설군과 지역 공통 혜택의 별도 안내
- 재사용 라이선스를 확인한 시설 사진과 저작자·라이선스 출처 표시
- 키보드 접근성과 모션 축소 설정 지원
- 경기도 영외마트 49곳 검색·지역 필터·SVG 매핑
- PX 상세 운영시간·주소·네이버지도·국군복지포털 연결

- PX 기준일: 2026-04-28
- PX 공식 원본: 국방부 국군복지단 영외마트 운영 현황
- PX 화면용 데이터: `src/data/px-stores-gyeonggi.json`

새 엑셀을 반영하려면 원본 파일을 교체한 뒤 변환 스크립트를 실행합니다.

```bash
npm install
npm run prepare:data
npm run prepare:map
```

변환 과정에서 공백과 결측 표시를 정리하고, `숙박·관광`을 화면 표기인 `숙박&관광`으로 바꾸며, 주소·전화·URL 사용 가능 여부와 검토 대상 행을 기록합니다.

## 주의 사항

- 시군 지도는 지역 선택용 행정경계입니다. 개별 시설 위치는 주소가 있을 때 네이버 지도 검색으로 연결합니다.
- 시설 사진은 현장 식별을 돕는 참고 이미지이며 현재 운영 상태를 보장하지 않습니다.
- 원본 주소가 없는 행은 지도 버튼을 표시하지 않습니다.
- 혜택과 운영 정보는 변경될 수 있으므로 실제 이용 전 시설 또는 담당 기관 확인이 필요합니다.
- PX 좌표는 도식 지도 표시용 후보값이며 정밀 길찾기에 사용하지 않습니다.
- PX 운영시간과 이용 자격은 매장 또는 국군복지포털에서 방문 전 재확인해야 합니다.
