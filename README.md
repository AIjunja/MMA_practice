# 병역명문가 혜택찾기

경기도 병역명문가 예우시설을 지역과 업종별로 찾는 모바일 우선 웹앱입니다.

## 주요 기능

- 경기도 30개 시군 지역 선택
- 7개 업종 필터
- 시설명·감면 내용 검색
- 우대 대상, 감면 내용, 비고 상세 조회
- 원본 데이터가 있는 경우 전화·홈페이지·네이버 지도 연결
- 주소가 없는 시설군과 지역 공통 혜택의 별도 안내
- 키보드 접근성과 모션 축소 설정 지원

## 데이터

- 기준일: 2026-04-30
- 원본: `data/source/facilities-2026-04-30.xlsx`
- 전체 레코드: 441건
- 변환 결과: `src/data/facilities.json`
- 품질 리포트: `data/quality-report.json`

새 엑셀을 반영하려면 원본 파일을 교체한 뒤 변환 스크립트를 실행합니다.

```bash
npm install
npm run prepare:data
```

변환 과정에서 공백과 결측 표시를 정리하고, `숙박·관광`을 화면 표기인 `숙박&관광`으로 바꾸며, 주소·전화·URL 사용 가능 여부와 검토 대상 행을 기록합니다.

## 배포

`main` 브랜치에 푸시하면 GitHub Actions가 테스트와 빌드를 실행한 뒤 GitHub Pages에 배포합니다.

예상 주소: `https://aijunja.github.io/MMA_practice/`

## 주의 사항

- 지도 좌표는 아직 포함하지 않았습니다. 주소가 있는 시설은 네이버 지도 검색으로 연결합니다.
- 원본 주소가 없는 행은 지도 버튼을 표시하지 않습니다.
- 혜택과 운영 정보는 변경될 수 있으므로 실제 이용 전 시설 또는 담당 기관 확인이 필요합니다.
