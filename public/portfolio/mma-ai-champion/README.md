# MMA AI Champion Portfolio Assets

병무청 AI 챔피언 교육 현장에서 도출한 요구사항을 구현한 공개 레퍼런스의 포트폴리오 이미지다.

- `wireframe-workshop.jpg`: 공무원 참여자와 함께 정리한 종이 와이어프레임
- `mobile-home.png`: 경기도 31개 시군 예우시설 탐색 첫 화면의 포트폴리오용 상단 캡처
- `mobile-facility-detail.png`: 예우시설 상세정보 상단 캡처
- `mobile-px-detail.png`: 경기도 영외마트 상세정보 상단 캡처
- `mobile-*-full.png`: 같은 화면의 전체 길이 검증 캡처

서비스 화면은 `scripts/capture-portfolio-screens.py`로 캡처한다. 브라우저의 모션과 전환 효과를 끄고 화면이 안정된 뒤 2배 해상도로 저장해, 중간 애니메이션의 낮은 불투명도가 이미지에 남지 않게 한다.

개인 이름, 얼굴, 연락처와 녹음 원문은 포함하지 않는다. 이 결과물은 병무청 공식 서비스나 기관 도입 사례가 아니라 교육 현장의 요구를 바탕으로 만든 공개 레퍼런스다.
